== Description ==

Allows you to manually set which JavaScript files to optimize with the CloudFlare Rocket Loader.

== Installation ==

Installation is easy:

* Download and unzip the plugin.
* Copy the folder to the plugins directory of your blog.
* Enable the plugin in your admin panel.
* An options panel will appear under Settings.
